How It Works
01 · SCORE

Composite Momentum

Each asset's price return is measured over three windows — 3, 6 and 12 months — then averaged into a single composite score. Equal-weight across all horizons.

02 · RANKING

Why Three Windows?

Short-term (3M) catches acceleration; medium-term (6M) confirms the trend; long-term (12M) filters noise and validates structural outperformance.

03 · SELECTION

Top 3 Allocation

The three highest-scoring asset classes are highlighted as the current month's allocation targets. Rebalance on the 20th when the new rankings publish.

04 · DATA

Source & Automation

Prices fetched via Yahoo Finance through a GitHub Actions workflow. The script runs automatically every 20th of the month and commits updated JSON data.

Composite Score = ( Return3M + Return6M + Return12M ) ÷ 3
